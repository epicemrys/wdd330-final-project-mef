# Marriage Event Finder

A simple web app to find local marriage-related events using Ticketmaster API and Google Maps.

## Setup
1. API keys: Ticketmaster (developer.ticketmaster.com), Google Maps (console.cloud.google.com).
2. Open in VS Code, use Live Server to run.

## Features
- Search events by keyword/city.
- View details with map.
- Save favorites (localStorage).
- Responsive design.

Made with HTML/CSS/JS. Deployed to GitHub Pages.
